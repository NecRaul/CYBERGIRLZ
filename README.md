# CYBERGIRLZ

Nonna (and MAGA) approved Discord bot.

![MAGA 2024](MAGA2024.jpg)

## How to run it

Replace the token in `config.json` with your bot's token and paste your YouTube cookies to `cookies.json`.

```Bash
node index.js
```

## Commands

* `!nonna`
* `!shadilay`
* `!ymca`

## Bot permissions

* Read Messages/View Channels
* Connect
* Speak
